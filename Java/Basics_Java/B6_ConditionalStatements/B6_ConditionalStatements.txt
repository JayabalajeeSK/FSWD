‚úÖ 1. IF Statement
‚úî What is it?

The simplest conditional statement.
Executes a block only when condition is true.

‚úî When to use?

When you want to run code for one specific condition.

‚úî What can be the condition?

boolean expression

comparison (==, >, <, etc.)

method returning boolean

logical operators (&&, ||)

‚úî Example Program
public class IfDemo {
    public static void main(String[] args) {
        int age = 20;

        if (age >= 18) {
            System.out.println("Eligible to vote");
        }
    }
}

‚úÖ 2. IF‚ÄìELSE Statement
‚úî What is it?

Provides two-way decision.

‚úî When to use?

When you want either this OR that.

‚úî Example Program
public class IfElseDemo {
    public static void main(String[] args) {
        int marks = 50;

        if (marks >= 40) {
            System.out.println("Pass");
        } else {
            System.out.println("Fail");
        }
    }
}

‚úÖ 3. Nested IF
‚úî What is it?

IF inside another IF.

‚úî When to use?

When conditions depend on multiple levels.

‚úî Example Program
public class NestedIfDemo {
    public static void main(String[] args) {
        int age = 25;
        boolean hasLicense = true;

        if (age >= 18) {
            if (hasLicense) {
                System.out.println("You can drive");
            }
        }
    }
}

‚úÖ 4. Else-IF Ladder
‚úî What is it?

Used for multiple conditions, only one block runs.

‚úî When to use?

When you have more than two conditions such as grading, salary slabs, menu selection.

‚úî Example Program
public class ElseIfLadderDemo {
    public static void main(String[] args) {

        int marks = 85;

        if (marks >= 90) {
            System.out.println("Grade A");
        } else if (marks >= 75) {
            System.out.println("Grade B");
        } else if (marks >= 50) {
            System.out.println("Grade C");
        } else {
            System.out.println("Fail");
        }
    }
}

‚úÖ 5. Switch Case

Java supports switch with:

int

byte

short

char

String

Enum

Wrapper types (Integer, Character)

final variables

‚úî When to use?

Many conditions with exact match

Cleaner than else-if ladder

Menu-based programs

‚≠ê Switch with int
public class SwitchIntDemo {
    public static void main(String[] args) {
        int day = 3;

        switch (day) {
            case 1: System.out.println("Monday"); break;
            case 2: System.out.println("Tuesday"); break;
            case 3: System.out.println("Wednesday"); break;
            default: System.out.println("Invalid");
        }
    }
}

‚≠ê Switch with char

üî• ALL Conditional Statements in ONE Code (Premium Interview Version)
public class AllConditionDemo {

    public static void main(String[] args) {

        int number = 75;

        // IF
        if (number > 0)
            System.out.println("Positive Number");

        // IF-ELSE
        if (number % 2 == 0)
            System.out.println("Even");
        else
            System.out.println("Odd");

        // NESTED IF
        if (number > 50) {
            if (number < 100) {
                System.out.println("Between 50 and 100");
            }
        }

        // ELSE-IF LADDER
        if (number >= 90)
            System.out.println("A Grade");
        else if (number >= 60)
            System.out.println("B Grade");
        else
            System.out.println("C Grade");

        // SWITCH CASE
        int option = 3;
        switch (option) {
            case 1: System.out.println("Option One"); break;
            case 2: System.out.println("Option Two"); break;
            case 3: System.out.println("Option Three"); break;
            default: System.out.println("Invalid Option");
        }
    }
}

üìù Top Interview Questions & Answers
Q1: Difference between if-else and switch?

if-else

For complex boolean conditions

Supports ranges

Supports relational/logical operators

switch

Only exact matching

Faster when many options

Supports int, char, String, Enum

Q2: Can switch handle ranges like 1‚Äì10?

‚ùå No
You must use if-else.

Q3: Can we use boolean in switch?

‚ùå No.
Java does NOT allow boolean in switch.

Q4: Why String is allowed in switch?

Since Java 7
‚Üí For cleaner code and fast string comparison using hashing.

Q5: Can we use expressions inside switch case labels?

Only constant expressions allowed.

final int B = 20;

    case A + B:  // allowed
}
