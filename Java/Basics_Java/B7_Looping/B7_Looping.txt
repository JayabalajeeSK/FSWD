1. FOR LOOP
‚úî What is it?

Loop with counter, used when the number of iterations is known.

‚úî When to use?

Run something for a fixed number of times

Arrays, indexing

Patterns

Timed operations

‚úî Syntax
for (initialization; condition; iteration) {
    // loop body
}

‚úî Example Program
public class ForDemo {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Count: " + i);
        }
    }
}

‚úÖ 2. WHILE LOOP
‚úî What is it?

Loop that runs as long as condition is true.

‚úî When to use?

Unknown number of iterations

Taking input until user stops

Running until a specific condition changes

‚úî Syntax
while (condition) {
    // loop body
}

‚úî Example Program
public class WhileDemo {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println("Number: " + i);
            i++;
        }
    }
}

‚úÖ 3. DO‚ÄìWHILE LOOP
‚úî What is it?

Loop that executes at least once, even if condition is false.

‚úî When to use?

Menu-driven programs

Login attempts

Input validation

When first iteration MUST run

‚úî Syntax
do {
    // statements
} while (condition);

‚úî Example Program
public class DoWhileDemo {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println("Value: " + i);
            i++;
        } while (i <= 5);
    }
}

‚úÖ 4. Enhanced For Loop (For‚ÄìEach)
‚úî What is it?

Loop for iterating over collections/arrays.

‚úî When to use?

When you don‚Äôt need index

For lists, arrays, sets, maps (keySet/values)

Easy iteration

‚úî Syntax
for (DataType var : arrayOrCollection) {
    // use var
}

‚úî Example Program
public class EnhancedForDemo {
    public static void main(String[] args) {
        int[] arr = {10, 20, 30};

        for (int n : arr) {
            System.out.println(n);
        }
    }
}

‚úÖ 6Ô∏è‚É£ Interview Questions (with Answers)
Q1: Why do we use nested loops?

Ans:
To handle 2D / multi-level structures like matrices, patterns, grids, combinations.

Q2: Time complexity of nested loops?

Ans:
If outer loop runs n and inner runs m
‚û° O(n √ó m)
If both run n
‚û° O(n¬≤)

Q3: Can we use break/continue inside nested loops?

Ans:
Yes.
break ‚Üí exits only current inner loop
continue ‚Üí skips to next iteration of current loop

Q4: Is nested for loop faster than nested while?

Ans:
Performance is the same; difference is readability and use case.

Q5: Can loops be infinitely nested?

Ans:
Yes, but practically up to 3‚Äì4 levels only, otherwise code becomes unmanageable.

Q6: Which loop is best for nested structures?

‚úî for ‚Üí patterns, fixed iterations
‚úî while ‚Üí when ending depends on condition
‚úî for-each ‚Üí arrays & collections

public class ForLoopsWithOutput {

    public static void main(String[] args) {

        System.out.println("A. Normal For");
        for (int i = 1; i <= 3; i++)
            System.out.println(i);

        System.out.println("\nB. Init Outside");
        int a = 1;
        for (; a <= 3; a++)
            System.out.println(a);

        System.out.println("\nC. Condition Missing");
        for (int i = 1; ; i++) {
            if (i > 3) break;
            System.out.println(i);
        }

        System.out.println("\nD. Increment Inside");
        for (int i = 1; i <= 3; ) {
            System.out.println(i);
            i++;
        }

        System.out.println("\nE. Multiple Variables");
        for (int i = 1, j = 3; i <= 3; i++, j--)
            System.out.println(i + " " + j);

        System.out.println("\nF. Decrementing Loop");
        for (int i = 3; i >= 1; i--)
            System.out.println(i);

        System.out.println("\nG. Function-Based Loop");
        for (int i = getStart(); i <= getLimit(); i = getNext(i))
            System.out.println(i);
    }

    static int getStart() { return 1; }
    static int getLimit() { return 3; }
    static int getNext(int i) { return i + 1; }
}

public class WhileLoopsWithOutput {

    public static void main(String[] args) {

        System.out.println("A. Normal While");
        int a = 1;
        while (a <= 3) {
            System.out.println(a);
            a++;
        }

        System.out.println("\nB. While with Break");

            System.out.println(b);
            b++;
        }

            i++;
            j--;

        int x = getStart();
        while (isValid(x)) {
            System.out.println(x);
            x = getNext(x);
        }
    }

    static int getStart() { return 1; }
    static boolean isValid(int x) { return x <= 3; }
    static int getNext(int x) { return x + 1; }
}
