8. STRINGS ‚Äî COMPLETE NOTES (With Code & Interview Questions)
1. String Class Basics
‚úî Meaning

String is a class in java.lang package, used to store sequence of characters.

‚úî Ways to create a String
1. Using String literal

Stored in String Constant Pool

String s1 = "Hello";
String s2 = "Hello"; // same object reused

2. Using new keyword

Stored in Heap

String s3 = new String("Hello");

‚úî Why Strings are widely used?

Used in input/output

Used in file reading

Used in network communication

Used in database queries

2. String Constant Pool (SCP)
‚úî What is SCP?

A special memory area inside Heap where Java stores string literals.

‚úî Example:
String a = "Java";
String b = "Java";

System.out.println(a == b); // true (same memory)

‚úî But using new:
String c = new String("Java");
System.out.println(a == c); // false (different objects)

3. Immutability of String
‚úî Strings cannot be changed once created.

Every modification creates a new object.

‚úî Why immutable?

Thread-safe

Security (passwords, URLs)

Used in HashMap keys

Better performance in SCP

‚úî Example:
String s = "Hello";
s = s + " World"; // new object created

4. IMPORTANT STRING METHODS (With Code)

I will include all major ones.

1. length()
String s = "Java";
System.out.println(s.length()); // 4

2. charAt(index)
String s = "Java";
System.out.println(s.charAt(2)); // v

3. substring()
String s = "programming";
System.out.println(s.substring(3));    // gramming
System.out.println(s.substring(3, 7)); // gram

4. contains()
String s = "Java Programming";
System.out.println(s.contains("Java")); // true

5. equals() and equalsIgnoreCase()
String a = "Hello";
String b = "hello";

System.out.println(a.equals(b));            // false
System.out.println(a.equalsIgnoreCase(b));  // true

6. startsWith() / endsWith()
String s = "hello.java";
System.out.println(s.startsWith("hello")); // true
System.out.println(s.endsWith(".java"));  // true

7. indexOf(), lastIndexOf()
String s = "banana";
System.out.println(s.indexOf("a"));       // 1
System.out.println(s.lastIndexOf("a"));   // 5

8. toLowerCase(), toUpperCase()
System.out.println("Java".toLowerCase());

9. trim()
String s = "   test   ";
System.out.println(s.trim()); // test

10. replace()
System.out.println("Java".replace('a', 'o')); // Jovo

11. split()
String s = "a,b,c";
String[] arr = s.split(",");

12. join()
String result = String.join("-", "A", "B", "C"); // A-B-C

‚úî Code:

6. StringBuilder (Mutable + NOT Synchronized)
‚úî Key Points

Mutable

NOT thread-safe

Fastest performance

‚úî Code:

7. String.format()
‚úî Used to format strings similar to printf

Common placeholders:
