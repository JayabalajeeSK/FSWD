A Linked List is a data structure where elements are stored in nodes, and each node points to the next node.
Think of it like a chain where each link is connected to the next link.

A Linked List is a list where:
Each element has data and a pointer (or reference) to the next element
Elements are not stored continuously in memory like arrays
